## Завдання 1. 
Розташувати в порядку зростання ентропії:

1) 5w, 4b
2) 5w, 5b
3) 6w, 4b
4) 6w, 6b

Для оцінки ентропії застосовуватиму ентропію Шеннона
$$H(p_w, p_b) = -p_w \log p_w - p_b \log p_b$$

### 1. 5w, 4b
```
n = 5+4 = 9;
p_w = 5/9, p_b = 4/9;
H(5/9, 4/9) = -0.
```
